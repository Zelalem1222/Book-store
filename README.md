# Book-Store

> The Bookstore is a website allows to 

   - Display a list of books.
   - Add a book.
   - Remove a selected book.



## Built With

- HTML
- CSS 
- Javascript
- React

## Getting Started

To get a local copy up and running follow these simple example steps.

- Clone repository in your local machine 
- cd Book-store
- npm install
- npm start
